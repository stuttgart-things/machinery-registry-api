# machinery-registry-api

lightweight machinery inventory API

## Prerequisites

- Go 1.25.6+
- [Task](https://taskfile.dev/) (optional, for task automation)

## Getting Started

```bash
# Clone the repository
git clone https://github.com/stuttgart-things/machinery-registry-api.git
cd machinery-registry-api

# Install dependencies
go mod tidy

# Run the application
go run .
# or with Task
task run
```
